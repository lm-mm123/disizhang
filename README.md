# 高级数据可视化仪表板

一个功能丰富的响应式数据可视化仪表板，支持多种图表类型、主题定制和实时数据交互。

## 🌟 主要特性

- **📊 多种图表类型**：柱状图、折线图、饼图、散点图
- **📈 丰富的数据集**：图书采购、汇率走势、天气数据、销售数据
- **🎨 主题定制**：4种精美主题（现代、专业、深色、优雅）
- **⚡ 实时响应**：数据修改后图表自动更新
- **🎯 自定义样式**：颜色、线型、标记等样式参数可调
- **📱 响应式设计**：适配桌面和移动设备
- **💾 图表导出**：支持PNG格式导出

## 📊 数据实例

### 1. 图书采购数据
- **数据类型**：不同地区图书采购情况对比
- **支持图表**：柱状图、折线图、饼图
- **数据格式**：图书种类、地区1数据、地区2数据

### 2. 汇率数据
- **数据类型**：2017年和2019年美元/人民币汇率走势
- **支持图表**：散点图、折线图
- **数据格式**：日期、2017年汇率、2019年汇率

### 3. 未来15天气温
- **数据类型**：最高气温和最低气温对比数据
- **支持图表**：折线图、柱状图
- **数据格式**：日期、最高气温、最低气温

### 4. 产品季度销售额
- **数据类型**：四个产品在不同季度的销售数据
- **支持图表**：柱状图、折线图
- **数据格式**：季度、产品A-D销售额

## 🎨 主题选项

## 🛠️ 使用方法

### 基本操作
1. **选择数据源**：在顶部控制栏选择要展示的数据
2. **选择图表类型**：柱状图、折线图、饼图或散点图
3. **修改数据**：在左侧边栏直接编辑数据值
4. **自定义样式**：调整颜色、线型、标记等参数
5. **导出图表**：点击"导出图表"按钮保存为PNG

### 数据格式说明
- 使用逗号分隔的数值格式
- 支持整数和浮点数
- 标签使用逗号分隔的文本

## 🎯 技术栈

- **前端框架**：HTML5 + CSS3 + JavaScript
- **图表库**：Chart.js 3.x
- **字体**：Google Fonts (Inter, Roboto, Noto Sans SC)
- **布局**：CSS Grid + Flexbox
- **响应式**：Media Queries

## 📱 响应式特性

- **桌面端**：侧边栏+图表并排布局
- **平板端**：自适应调整布局
- **移动端**：单列布局，触摸友好

## 🔧 开发指南

### 添加新数据源
1. 在HTML中添加新的数据输入区域
2. 在JavaScript的 `initChart()` 函数中添加数据处理逻辑
3. 更新数据源选择器选项

### 添加新主题
1. 在CSS中定义新的主题类
2. 在 `changeTheme()` 函数中添加主题切换逻辑
3. 更新主题选择器选项

### 添加新图表类型
1. 在Chart.js中配置新的图表类型
2. 更新图表类型选择器
3. 添加相应的数据处理逻辑

## 📈 性能优化

- 使用防抖函数优化频繁的数据更新

- CSS变量实现主题切换，减少重绘
- 响应式设计减少不必要的渲染
